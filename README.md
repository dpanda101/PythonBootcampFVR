# PythonBootcampFVR
proyecto de IMC

Pasos para crear el proyecto formulario IMC. 
1. Entender de que se trataba el proyecto
2. Anotar en una hoja de papel lod datos que debo pedir en el formulario y que tipo de dato es cada uno.
3. En el caso de los número utilice float para tener un dato mas preciso aunque lei q se suele usar decimal para presición.
4. Abrir vsCode y crear un archivo con teminación .py para que sea ejecutable con python.
5. Empece con los elementos basicos del programa una vez que este funciono
6. Empece a buscar la manera de validacion de datos para evitar que el usuario meta otro tipo de datos como numeros en donde va strings o null
7. Valide que funcionara correctamente el programa
8. arme un repo de github y subi los archivos
9. Edite el read.me

Observaciones
A pesar que es un programa relativamente facil ayuda mucho para adaptarse a la sintaxis de python, entender la parte de formularios, e implementar codigo para evitar errores del parte de usuario puede que haya manera mas eficiente de hacerlo pero es mejor ir metiendo funciones al programa depende de lo que se desea hacer
